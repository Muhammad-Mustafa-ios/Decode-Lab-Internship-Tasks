# 📖 About

This repository contains the projects I completed during my **Decode Lab Python Internship**. Each task focuses on strengthening core Python concepts while solving practical programming problems through clean, organized, and reusable code.

These projects demonstrate my understanding of Python fundamentals, logical thinking, and console-based application development.

# 📂 Projects

| Task     | Project                      | Description                                                                                                             |
| -------- | ---------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| ✅ Task 1 | 📝 To-Do List                | A console-based task management application that allows users to add, view, update, and delete daily tasks efficiently. |

### Clone the Repository

```bash

```

### Navigate to the Project



Replace the filename with any other task you wish to execute.
